# Python-Group-Assignment-
Student Management System (Python terminal based)

A role-based academic management system built with Python to support daily operations within an educational institution.
The application enables administrators, teachers, students, and staff to manage courses, enrolments, schedules, attendance, grades, and institutional announcements in a centralized environment.

Overview

This system digitalizes common administrative and academic processes.
Instead of managing information manually, users interact with structured menus according to their role.

The platform emphasizes:

organized record keeping

consistent updates between roles

easy retrieval of academic information

System Roles

The system supports four types of users:

Admin

Teacher

Student

Staff

Each role has different permissions and accessible functions.

Application Workflow:

Launch the program.

Select a role.

Log in using authorized credentials.

Access the role-specific menu.

Perform operations.

Return to the main menu or exit.

**Features by Role**

Admin:

Admins maintain the overall structure of the institution.

Manage user accounts (create, delete, reset passwords)

View and update student records

Create, modify, or remove courses

Assign teachers to courses

Manage academic schedules

Generate institutional reports

Teacher:

Teachers manage teaching, assessment, and class organization.

Create and update courses

Plan lessons

Schedule weekly classes

Enrol or remove students

Record attendance

Enter grades and comments

Review student progress

Student:

Students access their own learning information.

Update personal profile

Enrol in subjects

View registered courses

Access course materials

Check grades

View timetable

Submit feedback

Read announcements

Staff:

Staff members handle operational and administrative data tasks.

Add and manage student information

Update or delete records

Process transfers and withdrawals

Assist in timetable management

Record institutional events
